📋 Regular Verb Conjugation Patterns

▼

-AR verbs (e.g. falar)

eu-o / -ei / -ava
você/ele/ela-a / -ou / -ava
nós-amos / -amos / -ávamos
vocês/eles/elas-am / -aram / -avam
Future: vou/vai/vamos/vão + falar

-ER verbs (e.g. comer)

eu-o / -i / -ia
você/ele/ela-e / -eu / -ia
nós-emos / -emos / -íamos
vocês/eles/elas-em / -eram / -iam
Future: vou/vai/vamos/vão + comer

-IR verbs (e.g. partir)

eu-o / -i / -ia
você/ele/ela-e / -iu / -ia
nós-imos / -imos / -íamos
vocês/eles/elas-em / -iram / -iam
Future: vou/vai/vamos/vão + partir

* In Brazilian Portuguese speech, the future tense is almost always expressed as ir + infinitive (e.g. "vou falar", "vai comer", "vamos partir"). The simple future conjugation is mostly written/formal.
